= Implementa configurazioni di sistema Cloud Volumes ONTAP ben strutturate
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Grazie alle analisi di configurazione e ai suggerimenti, sfrutta NetApp per implementare le best practice per i tuoi sistemi Cloud Volumes ONTAP in esecuzione su storage Microsoft Azure. Puoi facilmente verificare lo stato dell'architettura, scoprire eventuali problemi nelle tue configurazioni e ottenere consigli per migliorare l'architettura di qualsiasi sistema che non sia ottimizzato per affidabilità, sicurezza, efficienza, prestazioni e costi.

link:configuration-analysis-cvo.html["Scopri l'analisi della configurazione e lo stato ben architettato."]



[[about-this-task]]
== Informazioni su questa attività

La dashboard well-architected analizza quotidianamente le configurazioni di distribuzione del sistema Cloud Volumes ONTAP su Microsoft Azure. L'analisi giornaliera fornisce lo stato well-architected, approfondimenti e raccomandazioni.

La connettività tramite Link o agente Console consente alla dashboard ben strutturata nella NetApp Console di eseguire una scansione completa dei tuoi sistemi Cloud Volumes ONTAP per individuare problemi di prestazioni, protezione dei dati e configurazioni. Questo ti assicura di ricevere informazioni accurate e raccomandazioni per ottimizzare i tuoi sistemi Cloud Volumes ONTAP.

I requisiti di configurazione dello storage variano. Puoi escludere dall'analisi le configurazioni specifiche che non si applicano al tuo ambiente di storage. Questo ti aiuta a evitare avvisi non necessari e risultati di ottimizzazione imprecisi.



[[before-you-begin]]
== Prima di iniziare

* Devi link:https://console.netapp.com/add-working-environment/choose-type["aggiungi un sistema Cloud Volumes ONTAP"] accedere alla NetApp Console per visualizzare lo stato dell'architettura e i suggerimenti per i problemi di configurazione.
* link:https://docs.netapp.com/us-en/console-setup-admin/concept-agent-install-options.html["Dovresti distribuire un agente Console"] per l'analisi più completa delle risorse del tuo sistema.




[[view-recommendations-for-configuration-issues]]
== Visualizza i suggerimenti per i problemi di configurazione

Puoi visualizzare i suggerimenti per i problemi di configurazione dopo il completamento dell'analisi giornaliera. I suggerimenti spiegano le best practice e i potenziali rischi delle configurazioni non ottimizzate. Puoi anche scaricare un file CSV con i suggerimenti.

.Passaggi
. Accedi al link:https://console.netapp.com/home["NetApp Console"^].
. Dal menu NetApp Console, seleziona *Carichi di lavoro* e poi *Well-architected*.
. Dalla dashboard di Well-architected, seleziona *Visualizza* per qualsiasi configurazione.
+
Facoltativamente, seleziona image:button_download.png["Uno screenshot dell'icona di download"]per scaricare un file CSV con i consigli per tutte le configurazioni.

. Esamina la raccomandazione.


.Cosa c'è dopo
Risolvi i problemi di configurazione per mantenere i tuoi sistemi in funzione con prestazioni ottimali.
